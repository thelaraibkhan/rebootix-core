# `rebootix pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
rebootix pairing list whatsapp
rebootix pairing approve whatsapp <code> --notify
```
